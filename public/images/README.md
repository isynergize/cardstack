# Image Assets

Add images for each language to create culturally-relevant flashcards.

## Requirements

- **Filename**: `1.jpg`, `2.jpg`, ... `10.jpg`
- **Size**: 300x400px minimum (600x800px recommended for retina)
- **Format**: JPG or WebP
- **Content**: Each image should visually represent the quantity (1 item, 2 items, etc.)

## Theme Suggestions by Language

## Example

For Hawaiian (`haw/`):
```
haw/
├── 1.jpg   # 1 hibiscus flower
├── 2.jpg   # 2 sea turtles
├── 3.jpg   # 3 palm trees
├── 4.jpg   # 4 pineapples
├── 5.jpg   # 5 plumeria flowers
├── 6.jpg   # 6 surfboards
├── 7.jpg   # 7 shells
├── 8.jpg   # 8 tropical fish
├── 9.jpg   # 9 leis
└── 10.jpg  # 10 coconuts
```

## Fallback

If an image is missing or fails to load, the app automatically uses a placeholder image from picsum.photos.
